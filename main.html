About Weatherly

A portfolio-grade AI-powered weather and trip planning application.

Project Overview

Weatherly is a Machine Learning and AI-powered web application that combines real-time weather forecasting with intelligent trip planning. It uses the OpenWeatherMap API for weather data and provides AI-generated trip itineraries based on weather conditions, user preferences, and budget constraints. Built as a full-stack portfolio project demonstrating modern web development, data visualization, and AI integration.
